# WhatTfWeDoin

Collaborative Godot project repository.

## Getting Started

1. Install the same Godot version across the team (**Godot 4.6.2**).
2. Open this folder as a Godot project.
3. Create scenes/scripts and commit changes in small, clear units.

## Current Prototype Controls

- **Move**: WASD / Arrow keys / Left stick
- **Shoot**: Space / Joypad A
- **Focus (slow + show hitbox)**: Shift / Joypad LB

## Collaboration Notes

- Pull latest changes before starting work.
- Use feature branches for larger tasks.
- Write short, descriptive commit messages.
